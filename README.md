# 🚀 TechMaster Quiz App

A multi-language Quiz Application built using HTML, CSS, and JavaScript.

🔗 Live Demo:  
https://hema-0508.github.io/Techmaster-quiz/

---

## 📌 Features

- Login with username
- Select programming language (C, Python, Java, CSS, JavaScript)
- 15-second timer per question
- Progress bar
- Score calculation
- Top 5 Leaderboard
- Data stored using LocalStorage

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
